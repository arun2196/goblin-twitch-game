# Local Setup

## Requirements

- Node.js 20+
- Wrangler
- Cloudflare account

Install Wrangler:

```bash
npm install -g wrangler
```

## Clone

```bash
git clone https://github.com/arun2196/goblin-twitch-game.git
cd goblin-twitch-game
```

## Install

```bash
npm install
```

## Run locally

```bash
wrangler dev
```

## Deploy

```bash
wrangler deploy
```
